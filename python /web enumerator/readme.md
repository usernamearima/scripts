# Web Enumerator

Simple Python tool for basic web reconnaissance.  
It retrieves `robots.txt`, extracts links from `sitemap.xml`, and detects HTML forms on a target website.

## Features
- Fetch and display `robots.txt`
- Parse `sitemap.xml` and list URLs
- Detect HTML forms (action, method, inputs)
- Simple terminal report

## Requirements
Python 3

Libraries:
requests  
beautifulsoup4  
lxml  
colorama  

## Install dependencies:

pip install -r requirements.txt

## Usage

python main.py https://example.com
