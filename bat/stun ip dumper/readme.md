# STUN IP Dumper

A small Windows batch script using `tshark` to display IPv4 addresses from UDP/STUN traffic.

## Usage
- Run as **Administrator**
- Select the network **interface number**
- Stop with **CTRL + C**

## Requirements
- Windows
- Wireshark (`tshark`)

⚠️ Educational use only. Use on networks you own or have permission to test.
